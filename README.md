**Análise de Dados – Sample Superstore**

  Mini-projeto de análise de dados desenvolvido em Python a partir do dataset SampleSuperstore.
  O foco do projeto é praticar análise exploratória de dados e responder perguntas de negócio usando dados reais.

  O código foi escrito no VS Code, utilizando a extensão Jupyter, com células organizadas por blocos (# %%).

**Objetivo**

  Aplicar análise de dados para entender desempenho de vendas, lucro e prejuízo, simulando um cenário empresarial real.

  O projeto também faz parte do meu portfólio e do meu processo de aprendizado.

**O que foi analisado**

  Análise geral de vendas e lucro
  Margem de lucro
  Desempenho por região e estado
  Categorias e subcategorias mais e menos lucrativas
  Impacto dos descontos no lucro
  Ticket médio e lucro médio por venda
  Vendas com lucro negativo
  Vendas altas com prejuízo
  Comparação entre modos de envio (Ship Mode)

**Tecnologias utilizadas**

  Python 3
  Pandas
  VS Code com extensão Jupyter
  
  Dataset: SampleSuperstore.csv

**Como executar**

  Clone o repositório:
  
  git clone https://github.com/Tutusr32/Analise-de-Dados


Abra o projeto no VS Code
Certifique-se de ter as extensões Python e Jupyter instaladas
Execute as células sequencialmente para visualizar as análises

**Observações**

  A análise de outliers extremos de prejuízo foi discutida, mas não implementada em código, pois foge do meu nível atual de estudo.
  
  O foco do projeto é análise exploratória e interpretação de dados, não estatística avançada.
